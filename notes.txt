# SIGNUP + LOGIN
1. Add " gem 'devise' " to Gemfile
2. bundle install
3. rails g devise:install
4. Follow the steps to show alerts
5. rails g devise User
6. rails g devise:views

7. rails g controller pages home - create a home page
8. change "get 'pages/home'" to "root to: 'pages#home'"
9. rails db:migrate
